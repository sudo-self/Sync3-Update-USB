Sync3 Update USB macOS

Download the official update

Enter your VIN number https://www.ford.com and download the update file for your vehicle.
To keep things easy, rename your download to update.zip

Step 1: USB Setup

Open Terminal and list the available disks using the diskutil list command to identify the correct device identifier for your USB drive.

diskutil list

Step 2: Format USB to exFAT

Replace /dev/disk2 with the correct device identifier obtained from the previous step.

diskutil eraseDisk ExFAT MyUSB /dev/disk2

Step 3: Mount USB Drive

Mount the USB drive. Replace /dev/disk2s1 with the correct partition identifier.

diskutil mount /dev/disk2s1

Step 4: Copy Zip File to USB Drive

Copy the update.zip file to the USB drive. Ensure the path to the zip file is correct. /Volumes/MyUSB is an example mount point.

cp ~/update.zip /Volumes/MyUSB

Step 5: Unzip Contents to USB Drive

Unzip the contents of the zip file to the USB drive.

unzip /Volumes/MyUSB/update.zip -d /Volumes/MyUSB

Step 6: Eject USB Drive

Finally, eject the USB drive.

diskutil eject /dev/disk2

Insert the USB and Sync3 will automatically update

Terminal Commands

  • diskutil list - List all disks and their partitions.
  • diskutil eraseDisk ExFAT MyUSB /dev/disk2 - Format the disk to ExFAT. Replace /dev/disk2 with the correct disk identifier.
  • diskutil mount /dev/disk2s1 - Mount the USB drive. Replace /dev/disk2s1 with the correct partition identifier.
  • cp ~/update.zip /Volumes/MyUSB - Copy the update.zip file to the USB drive. Adjust the path if necessary.
  • unzip /Volumes/MyUSB/update.zip -d /Volumes/MyUSB - Unzip the contents to the USB drive.
  • diskutil eject /dev/disk2 - Eject the USB drive.
